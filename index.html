    Setup

    When the programm is installed you have to configure the exports and keys directories. These directories are where the saved key and crypt are located.
    Go to File > Settings enter the Default export location and Default key location fields. These fields correspond of the directories where the software will save the keys and files.

    You need to restart the software the see the changes.

    2: RSA

    How to use

    RSA is an algorythm wich use asymetrics keys to encrypt and decrypt messages. This software allow to encrypt and decrypt messages using privates and publics keys. But it does not allowed to produce RSA signatures. Suppossing two personn Alice and Bob. Alice want to recieve secret messages from Bob. So Alice generate two keys, the public and the private key. Alice give the public key to Bob, which how use it to encrypt his message. Bob transmit is encrypted message to Alice, which one decrypt it with the private Key.

    Generate keys

    To generate RSA keys just click on the button "Generate RSA Keys" in the title screen. Select your key size (4096 or 2048 bits) and click to the generate button. The public and private keys will appears in the corresponding fields. To used it you have to downloaded it by clicking on the corresponding save button .

    Encrypt a message

    To encrypt your secret message using RSA algorythm, click on "Encrypt a Message" button on the title screen.
    First select your public key, you can write the file's location in the text field, or select it with the search button . Then click on "validate" to validate your key. If all is correct the encrypt area will be unlocked. Enter your secret message in the text area.
    Be aware: RSA algorythm is used for short messages, the max length correspond to length of the key (4096/2048).
    Once you have witten your secret message click on the "Encrypt" button. The encrypted message will appears in the last text area. You can download the encrypted message by clicking on the save button .
    Once everything done, click on the "Reset" button.

    Decrypt a message

    To decrypt a secret message using RSA algorythm, click on "Decrypt a Message" button on the title screen.
    First select your private key, you can write the file's location in the text field, or select it with the search button . Then click on "validate" to validate your key. If all is correct the decrypt area will be unlocked.
    Enter encrypted message in the text area. Once you have witten your encrypted message click on the "Decrypt" button. The secret message will appears in the last text area. You can download the decrypted message by clicking on the save button .
    Once everything done, click on the "Reset" button.

    3: AES

    How to use

    AES algorythm is use for encrypt file with a single secured key. In opposition of RSA algorythm, you use the same key for encyption and decryption.

    Generate key

    To generate AES keys just click on the button "Generate AES Keys" in the title screen. Select your key size (128, 192 or 256 bits) and click to the generate button.
    The key will appears in the result fields. To used it you have to downloaded it by clicking on the save button .

    Encrypt a file

    To encrypt your file using AES algorythm, click on "Encrypt a File" button on the title screen.
    First select your key, you can write the file's location in the corresponding text field, or select it with the search button .
    Then select the file you want to encrypt, by enter it's location in corresponding text field, or select it with the search button .
    Once you have for key and file, click on the "Encrypt" button. It will encrypt your file and save it to the exports directories choose in the settings.

    Decrypt a file

    To decrypt your encrypted file using AES algorythm, click on "Decrypt a File" button on the title screen.
    First select your key, you can write the file's location in the corresponding text field, or select it with the search button .
    Then select the file you want to decrypt, by enter it's location in corresponding text field, or select it with the search button .
    And for the last step, choose the output name of the decrypted file (do not forget the file extension).
    Once you have for key, input file and output file name, click on the "Decrypt" button. It will decrypt your file and save it to the exports directories choose in the settings.
